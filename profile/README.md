### 📦 Projects

| Repo | Description | Stack |
|---|---|---|

### 🛠️ How We Work

- Every feature is built on its own branch, merged via reviewed Pull Requests
- Issues are tracked and labeled (`bug`, `enhancement`, `good first issue`)
- CI runs lint + tests on every PR
- Releases follow [semantic versioning](https://semver.org/) once a project hits a stable milestone
